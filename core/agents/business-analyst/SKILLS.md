# Business Analyst — Available Skills

Skills are invoked on demand. Each skill file is self-contained and can be composed into larger workflows.

| Skill | File | When to use |

| requirements-gathering | @core/skills/business-analyst/requirements-gathering.md | Extract and document requirements from stakeholders |

| prd-writing | @core/skills/business-analyst/prd-writing.md | Produce a product requirements document for a feature or initiative |
| process-analysis | @core/skills/business-analyst/process-analysis.md | Map and analyse an existing business process to find gaps or improvements |
| roadmap | @core/skills/business-analyst/roadmap.md | Build or update a prioritised product roadmap |
| sprint-planning | @core/skills/business-analyst/sprint-planning.md | Prepare and facilitate a sprint planning session |
| estimation | @core/skills/business-analyst/estimation.md | Estimate effort or complexity for a backlog of work |
| innovation-discovery | @core/skills/business-analyst/innovation-discovery.md | Explore unspoken needs and opportunities before a ticket exists |
